# Датасеты

Здесь будем хранить материалы по каждому датасету отдельно.

Внутри папки конкретного датасета будут лежать входные требования, критерии/оценки, тест-кейсы, результаты прогонов и заметки по ручному разбору.

## Текущие датасеты

- `customer_gold_v1/` - высокоприоритетный эталонный датасет от заказчика: требования и эталонные ТК.
- `cpt_smoke_v1/` - первый пример MVP, соответствует `CASE-001` из `imports/example_dataset_case.json`.
