# customer_gold_v1

Высокоприоритетный эталонный датасет от заказчика.

## Назначение

Здесь храним эталонные требования и эталонные тест-кейсы, которые считаются наиболее критичными для проекта.

На текущем этапе заведены эталонные требования по интеграции С.ФТ с С.Релизы. Всем требованиям в базе выставлен score `10.0` по 10 критериям качества требований.

## Содержимое

- `requirements/` - эталонные требования от заказчика.
- `test-cases/` - эталонные тест-кейсы от заказчика.
- `source/` - исходные файлы без изменений.
- `notes.md` - рабочие заметки по датасету.
