# customer_gold_v1

Высокоприоритетный эталонный датасет от заказчика.

## Назначение

Здесь храним пример эталонных требований и эталонных тест-кейсов, которые считаются наиболее критичными для проекта.

На первом этапе материалы только сохраняем. Оценку по критериям и загрузку в базу выполняем отдельно, когда договоримся о формате и начнем разбор.

## Содержимое

- `requirements/` - эталонные требования от заказчика.
- `test-cases/` - эталонные тест-кейсы от заказчика.
- `source/` - исходные файлы без изменений.
- `notes.md` - рабочие заметки по датасету.
