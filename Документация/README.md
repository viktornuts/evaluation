# Документация

Здесь лежат проектные документы и рабочие материалы по датасетам.

## Структура

```text
Доки по проекту + критерии/
  Общий контекст проекта, план, глоссарий, критерии и исходные документы.

Датасеты/
  Материалы по каждому датасету: входные требования, оценки, тест-кейсы и заметки по разбору.
```
